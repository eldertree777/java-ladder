# 🚀 2단계 - 사다리(생성)
## 기능 목록
- User
  - [x] 이름이 null, 공백 및 빈값인 경우 예외 발생
  - [x] 이름은 최대 5글자이며, 초과시 예외 발생

- Users
  - [x] 유저들의 수를 반환
  - [x] 크기가 0이거나 비어있으면 예외 발생

- Point
  - [x] 자신의 상태를 반환 해야함.

- Line(가로 선) : Point의 List
  - [x] n-1 번째 이어진 경우 n번째는 이어질 수 없음.
  - [ ] 사람 수 만큼 사다리 폭이 결정된다.
  - [x] 사다리 폭은 1보다 작은 경우 예외 발생

- Ladder : Line의 List
  - [x] 높이가 1보다 작은 경우 예외 발생

## 프로그래밍 요구사항
- 스트림과 람다를 적용
- 모든 엔티티를 작게 유지
